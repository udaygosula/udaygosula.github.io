Storage Classes

  • Storage Class specifiers tells the compiler about location to store variable, scope of the variable, default initial value of the variable and life time of the variable.
  1. Location : It specifies where the variable to be stored either CPU or register.
  2. Default initial value : It specifies the intital value that is assigned to the variable without explicitily initialized.
  3. Scope : It specifies the availability of a perticular variable inside or ouside the block or program.
  4. Lifetime : It specifies how long a variale can exist.
Storage Classes Location Default Initial Value Scope Life - time
auto RAM Garbage Value Local Within function or block.
extern RAM Zero Global Till the end of main program, May be declared anywhere in the program.
static RAM Zero Local Till the end of main program, retains value between multiple functions call.
register Register Garbage Value Local Within function.

Note :
     By default the storage class will be auto.
