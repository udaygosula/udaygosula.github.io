Pointers

  • Pointer is a variable that stores/points the address of another data item(variable or an array element) of same data type.
  • Pointer is used to allocate memory dynamically(at run time).
Note :
    When a variable is defined, a memory location is assigned for it, in which it's value will be stored.
    To check the address of any variable use '&' with variable name.
syntax :
    &var_name;
Example :
    int a = 10;
    printf("The value of a : %d\nThe address of a : %u",a,&a);
Output :
    The value of a : 10
    The address of a : fff4

Pointer Operations :

There are four different types of operations performed on poibters
  • Incrementing/Decrementing a pointer
  • Addition/Subtraction of a constant number to a pointer
  • Subtraction of one pointer from another
  • Comparison of two pointers
Opeator Example
++/-- int a=5,*x; x=&a; x++;
+(constant) int a = 10, *x, y; x = &a; y = x + 5
-(pointer) int a = 10, b = 20, *x, *y; x = &a; y = &b; x-y;
=/!= int a = 10, *x *y; x = &a; y = &a; if(x == y) printf("Both are equal");


Pointer to array :
  • we can use a pointer to point to an array, and then we can use that pointer to access the array elements.
Example :
    int i;
    int a[5] = {1, 2, 3, 4, 5};
    int *p = a;
    /* same as int*p = &a[0] */
    for (i=0; i<5; i++)
    {
        printf("%d ", *p);
        p++;
    }
Output :
    1 2 3 4 5
Note :
  • printf("%d",a[i]); ----- prints the array, by incrementing index
  • printf("%d",i[a]); ----- prints the array, by incrementing index
  • printf("%d",a+i); ------ prints the address of array elements by incrementing index
  • printf("%d",*(a+i)); ----- prints the array, by incrementing index
  • printf("%d",*a); ------ prints the starting element of array, i.e, a[0]

Dynamic memory allocation :

  • Dynamic memory allocationto is to allocate memory at runtime.
  • Dynamic memory allocation is possible by 4 functions of stdlib.h header file.
  • The four functions used are
    1. malloc()
    2. calloc()
    3. realloc()
    4. free()

Functions used in dynamic memory allocation :
function description syntax example
malloc() It allocates single block of requested memory.
It doesn't initialize memory at execution time, so it has garbage value initially.
It returns NULL if memory is not sufficient.		 ptr = (data_type*) malloc (byte-size); int n = 10, *ptr;
ptr = (int*) malloc (n*sizeof(int));
calloc() It allocates multiple block of requested memory.
It initially initialize all bytes to zero.
It returns NULL if memory is not sufficient.		 ptr=(data_type*) calloc (number, byte-size); int n = 10, *ptr;
ptr = (int*) calloc (n,sizeof(int));
realloc() It reallocates the memory occupied by malloc() or calloc() functions. It changes the memory size. ptr = realloc (ptr,new_size); int n = 10, *ptr;
ptr = (int*) calloc (n,sizeof(int));
ptr = (int*)realloc(ptr,20);
free() It frees the dynamically allocated memory. free(ptr) int n = 10, *ptr;
ptr = (int*) calloc (n,sizeof(int));
